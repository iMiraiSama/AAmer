# AAmer Platform

**AAmer** is a web-based platform developed as a capstone project at Prince Sultan University. It connects users with professional and verified service providers across various industries in Saudi Arabia.

## Project Overview

AAmer aims to simplify the process of requesting services such as home cleaning, tailoring, tutoring, and more. Users can submit service requests and get matched with trusted providers based on filters like location, budget, and rating.

---

## Key Features

- User registration & secure login  
- Service request posting and provider offer system  
- Real-time chat and booking confirmation  
- Provider profiles with reviews and dynamic ratings  
- Payment integration (Card & Cash)
- Filtering and search enhancements

---

## Technology Stack

- **Frontend:** HTML, CSS, JavaScript  
- **Backend:** Node.js, Express  
- **Database:** MongoDB  
- **Authentication:** Firebase  
- **Notifications & Messaging:** Email API + custom chat  
- **Deployment:** GitHub
